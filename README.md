# Water Park Operations Simulation
## Stochastic Modeling & Discrete Event Simulation using Python

### Project Overview
This project focuses on optimizing the operational efficiency of a water park using **Discrete Event Simulation (DES)**. The simulation models visitor flow and waiting times at various attractions (Water Slides) to identify bottlenecks and improve service levels.

### Key Features & Contributions:
* **Stochastic Modeling:** Modeled visitor arrivals and service times using statistical distributions based on real-world data.
* **Simulation Framework:** Developed a full-scale simulation model using **Python** and the **SimPy** library.
* **Data-Driven Insights:** Analyzed distribution data from large and small tube slides to optimize resource allocation.
* **Performance Metrics:** Evaluated system performance based on average wait times, resource utilization, and queue lengths.

### Tech Stack:
* **Language:** Python
* **Libraries:** SimPy (Simulation), Pandas (Data Manipulation), Matplotlib/Seaborn (Visualization).
* **Environment:** Google Colab / Jupyter Notebook.

### How to Run:
1. Ensure you have the datasets in the `/Data` folder.
2. Install dependencies: `pip install simpy`
3. Run the `Simulation_Model.ipynb` notebook to see the results and visualizations.
